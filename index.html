Online Calculator

This project was built as a way to learn JavaScript and how it interacts with HTML and CSS. It contains lots of useful features over a standard calculator such as: powers, roots, positive and negative signs, keyboard inputs, etc. While the actual calculations are not complicated, the way all the elements work together and the way the calculator display updates taught me alot about UI.

Soduko Solver

This is a project I made using JavaScript, CSS and HTML (originally made in C++) to solve the classic game of Sudoku! It uses a backtracking algorithm to move through the puzzle and find the correct solution. I plan on adding more features soon as I have time inbetween school work, which you can see on my github!
